# Speech analyzer

Extracts formants from video

## Tech
python 3.8.6
with libraries:
* matplotlib 3.3.3
* scipy 1.6.0
* moviepy 1.0.3
* cv2 4.5.1

## Installation
```sh
pip3 install matplotlib
pip3 install scipy
pip3 install moviepy
pip3 install opencv-contrib-python
```

## Usage
```sh
python3 main.py video.mp4
```
